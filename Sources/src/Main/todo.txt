* GFX
	- shader - содержит в себе ссылку на текстуры, матрицы и прочие константы. Замена всяким RS/TSS/VS/PS
	- необходимо уметь объединять shader'ы в некие MIP уровни для упрощения/усложнения эффекта с расстоянием/возможностями

* Optimization
	- удалить из сцены множество area maps и оставить только static objects, dynamic objects, units
	- при выводе террейна сначала формировать большой буффер, а потом выводить его через temp vertices/indices. 
		если террейн не изменяется в течении нескольких кадров, то создавать для него статический буфер
	- для рек и 3D дорог - нарезать их на strip'ы
	- при генерации партиклов по пути полёта оптимизировать выражение рассчёта положения

* Startup & API Checks
	- иметь список "обязательных" модулей и после всей загрузки проверить его
	- для модулей сделать "checker" - то, что проверяет наличие нужного API
